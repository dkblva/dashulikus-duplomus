# 🚀 БЫСТРЫЙ СТАРТ

Пошаговое руководство для запуска админ панели за 5 минут.

## Требования

- ✅ Python 3.8+ установлен
- ✅ pip (обычно идет с Python)
- ✅ Telegram Account
- ✅ FormAPI запущен (http://localhost:5000)

## Шаг 1: Получить Telegram Bot Token (2 мин)

### 1.1 Открыть Telegram и найти @BotFather

### 1.2 Начать разговор
```
Отправить команду: /start
```

### 1.3 Создать новый бот
```
Отправить команду: /newbot

BotFather: Как назвать вашего бота? 
Ответ: AdminPanel (или любое имя)

BotFather: Выберите username для вашего бота (он должен заканчиваться на `bot`).
Ответ: admin_panel_bot (или admin_yourname_bot)

BotFather вернет ваш TOKEN:
5107391307:AAG1...
```

### 1.4 Скопировать полученный TOKEN
```
Пример: 5107391307:AAG1_5V7DJBKzl3PF_4I0X4-9S_I8Fq1t-E8
```

## Шаг 2: Получить Ваш Telegram ID (1 мин)

### 2.1 Открыть Telegram и найти @userinfobot

### 2.2 Отправить команду
```
Отправить: /start

Бот вернет ваш ID в виде:
Your user ID is: 506698213
```

### 2.3 Запомнить этот ID
```
Это нужно для ADMIN_CHAT_ID в config.py
```

## Шаг 3: Установить зависимости (1 мин)

### 3.1 Перейти в папку бота
```bash
cd dashulikus-duplomus-master/bot
```

### 3.2 Установить пакеты
```bash
pip install -r requirements.txt
```

Если ошибка — используйте:
```bash
python -m pip install --upgrade pip
pip install -r requirements.txt
```

## Шаг 4: Конфигурация (1 мин)

### 4.1 Открыть config.py

### 4.2 Заменить значения YOUR_TOKEN и YOUR_ID

**БЫЛО:**
```python
TOKEN = "8283611145:AAFfb_ETrxyIZajAxVIQrKr0OgaJAlg2vWw"
ADMIN_CHAT_ID = 506698213
```

**СТАНЕТ:**
```python
TOKEN = "5107391307:AAG1_5V7DJBKzl3PF_4I0X4-9S_I8Fq1t-E8"  # Ваш токен
ADMIN_CHAT_ID = 123456789  # Ваш ID
```

### 4.3 Проверить API URL

**ДОЛЖНО БЫТЬ:**
```python
API_BASE_URL = "http://localhost:5000"
```

Если API на другом хосте, измените URL:
```python
API_BASE_URL = "http://192.168.1.100:5000"  # Или IP адрес VPS
```

## Шаг 5: Запустить бот (1 мин)

### 5.1 Убедиться что API работает

Открыть в браузере:
```
http://localhost:5000/swagger
```

Если открылся Swagger UI — все хорошо ✅

### 5.2 Запустить бот

**Windows (PowerShell/CMD):**
```bash
python bot_extended.py
```

**Linux/Mac:**
```bash
python3 bot_extended.py
```

### 5.3 Ждать сообщения о старте
```
🤖 Бот запускается...
```

## Частые проблемы

### ❌ "API не ответил"

**Проверить:**
```bash
# 1. API запущен?
curl http://localhost:5000/api/applications

# 2. URL правильный в config.py?
# 3. Firewall не блокирует?
```

**Решение:**
```python
# В config.py увеличить timeout:
API_TIMEOUT = 30
```

### ❌ "Бот не отвечает на сообщения"

**Проверить:**
```bash
# 1. Токен правильный?
# 2. Интернет есть?
# 3. @BotFather разрешил боту команды?
```

**Решение:**
1. Проверить TOKEN в config.py
2. Перезагрузить бота (Ctrl+C, потом python bot_extended.py)

### ❌ "Module 'telebot' not found"

**Решение:**
```bash
pip install telebot
# или
pip install pyTelegramBotAPI
```

### ❌ Bot работает, но не видно в Telegram

**Решение:**
```bash
# 1. Убедиться что боту разрешены все права
# @BotFather → Мой бот → Показать → API Token

# 2. Если только создан бот, ждать 1-2 минуты

# 3. Убедиться что юзнейм правильный
# Должен заканчиваться на "bot"
```

## Следующие шаги

### Продвинутая конфигурация

1. **Использовать .env файл:**
```python
# В config.py:
import os
from dotenv import load_dotenv

load_dotenv()

TOKEN = os.getenv("TELEGRAM_TOKEN")
ADMIN_CHAT_ID = int(os.getenv("ADMIN_CHAT_ID"))
API_BASE_URL = os.getenv("API_BASE_URL", "http://localhost:5000")
```

2. **Запустить как Service (Linux):**
```bash
sudo systemctl enable admin-bot
sudo systemctl start admin-bot
```

3. **Использовать Docker:**
```bash
docker build -t admin-bot .
docker run --env-file .env admin-bot
```

## Лучшие практики

- Используйте /start для перезагрузки
